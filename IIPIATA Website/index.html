Welkom

Wat is het doel van deze website?

Het doel van de website is een online omgeving bieden voor mijn bordspel: Ontsnap uit de piramide.

Wat kun je op deze website vinden?

Op deze website kun je extra informatie vinden bij vragen die voorkomen in het bordspel. Dit kun je doen door een viercijferige code uit het bordspel in te vullen op de site. Je kunt hieronder ook het spel bestellen op het moment dat het je leuk lijkt. Dat doe je door het formulier hieronder in de vullen. Ook staat hier wat aanvullende informatie over de opdrachtgever van het bordspel, het RMO (Rijksmuseum van Oudheden) in Leiden.

Bordspel bestellen?

Lijkt het bordspel je leuk en wil je het bestellen? Dat kan! Als je op de knop hieronder drukt kom je bij een bestelformulier.
